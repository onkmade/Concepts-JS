# Function

Functions are used everywhere in web development:
handling button clicks, validating forms, toggling dark mode, fetching data, updating UI, running calculations, literally everything. Once you understand functions, JavaScript will suddenly feel more structured and powerful.

Basic structure of a function:
```javascript
function functionName(parameters) {
    // code to be executed
}
```

## Local vs Global Scope
1. Variables defined inside a function are **local** to that function and cannot be accessed outside of it.
2. Variables defined outside any function are **global** and can be accessed from anywhere in the code.
3. Functions can access global variables, but not vice versa.
### Example
```javascript
let globalVar = "I am global";
function myFunction() {
    let localVar = "I am local";
    console.log(globalVar); // Accessible
    console.log(localVar);  // Accessible
}
myFunction();
console.log(globalVar); // Accessible
console.log(localVar);  // Error: localVar is not defined
```
## Questions:

1. **Greet a user** – `greet(name)` prints `"Hello, name!"`.
2. **Add two numbers** – `add(a, b)` returns the sum.
3. **Check even or odd** – `isEven(num)` returns `true` or `false`.
4. **Square a number** – `square(num)` returns the square.
5. **Full name** – `fullName(first, last)` returns `"First Last"`.
6. **Multiply three numbers** – `multiply(a, b, c)` returns the product.
7. **Check voting eligibility** – `canVote(age)` returns `"Eligible"` if age ≥ 18, else `"Not eligible"`.
8. **Convert Celsius to Fahrenheit** – `toFahrenheit(celsius)` returns the converted temperature.
9. **Repeat a string** – `repeatStr(str, times)` returns the string repeated the given number of times.
10. **Find the maximum** – `maxOfThree(a, b, c)` returns the largest of three numbers.

These include **printing, returning values, conditional logic, and basic math operations**,
 giving you a variety of ways to practice writing functions.

# Function Expression
A function expression is when you create a function and store it inside a variable, so the function itself becomes a value. You then call the function using the variable name. Unlike a normal function declaration, a function expression is not hoisted, meaning you cannot call it before it is created in the code. Function expressions are useful when you want to pass functions as data, use them in callbacks, or create functions only when needed.

### Example
```javascript
const greet = function(name) {
    return "Hello, " + name + "!";
};

console.log(greet("Alice")); // Outputs: Hello, Alice!
```

### How Parameters and Arguments Work

**Parameters** are placeholders that get replaced by real values when the function is called.
So the function *receives* outside values through those parameters.
Those values (also called Arguments) can be strings, numbers, or even other functions.

```js
function ask(q, yes, no) { if (confirm(q)) yes(); else no(); }
ask("Agree?", showOk, showCancel);   // real values replace q, yes, no
```
